# AI Tutor — Learning Tutor

An AI-powered Android learning app that delivers daily micro‑lessons, interactive quizzes, and an intelligent chat tutor. Track your streak, review mistakes, and master any topic—one concept at a time.

> **Stack**: Kotlin · Jetpack Compose · Material 3 · Room · Retrofit · Moshi  
> **Architecture**: MVVM + Clean Architecture  
> **API**: [ai-tutor.sebastianmorales.sbs](https://ai-tutor.sebastianmorales.sbs/)

---

## Features

### 🎯 Goal Setup
Define **what** you want to learn and **how long** you can study each day. The app tailors every lesson to your chosen topic and commitment.

### 📚 Daily Lessons
Each day brings a new concept with:
- A clear, concise **explanation**
- **Key ideas** in bullet points
- A **practical example**
- A **quiz** to reinforce understanding

Lessons are generated by a remote AI service and adapt to your progress.

### 💬 AI Chat Tutor
Stuck on a concept? Ask the tutor questions in natural language. It responds with context‑aware explanations tied to your current topic.

### 📈 Progress Tracking
- **Streak** – how many days in a row you've studied
- **Lesson history** – completed sessions and scores
- **Mistakes review** – revisit incorrect answers and clear them once you've corrected your understanding
- **Concept mastery** – see your level for each concept
- **Completed topics** – track what you've finished

### 🌍 Localization
- English (default)
- Spanish (`values-es/`)

## Architecture

```
app/src/main/java/com/example/
├── api/                  # Retrofit client, API interface, DTOs
├── data/
│   ├── database/         # Room database
│   ├── models/           # Room entities (UserProfile, SessionProgress, Mistake)
│   └── repository/       # TutorRepository (local), RemoteLearningService (API), mock generators
├── domain/
│   ├── model/            # Domain models (Lesson, ChatMessage, QuizQuestion, CycleInfo)
│   ├── usecases/         # GetLessonUseCase, GetChatTutorReplyUseCase, GetStreakUseCase
│   └── LearningService   # Interface for data sources
└── ui/
    ├── components/       # 22 reusable Compose components
    ├── screens/          # GoalSetup, DailySession, ChatTutor, Progress
    ├── state/            # State holders per screen (Lesson, Chat, Quiz, Progress)
    ├── util/             # Utility functions (TextParsing)
    ├── theme/            # Material 3 theme (indigo palette)
    ├── TutorViewModel.kt
    └── TutorViewModelFactory.kt
```

The app follows **clean architecture** layers:
- **Domain** — pure Kotlin models and use‑cases (no Android dependencies)
- **Data** — Room for local persistence, Retrofit for remote API, Repository pattern
- **UI** — Jetpack Compose screens with extracted reusable components and state holders

## Getting Started

### Prerequisites

- Android Studio Ladybug (2024.2.1+) or newer
- JDK 17
- Android SDK 36
- A physical device or emulator running Android 7.0 (API 24) or higher

### Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/SebastianMoralesDuque/ai-tutor-app.git
   cd ai-tutor-app
   ```

2. **Configure the API endpoint** (optional)
   
   The app points to `https://ai-tutor.sebastianmorales.sbs/` by default.  
   To change it, edit `RetrofitClient.BASE_URL` in:
   ```
   app/src/main/java/com/example/api/RetrofitClient.kt
   ```

3. **Set up environment variables** (if using a local backend)
   ```bash
   cp .env.example .env
   # Edit .env with your values
   ```

4. **Build and run**
   ```bash
   ./gradlew assembleDebug
   ```
   Or open the project in Android Studio and press **Run**.

### Running tests

```bash
./gradlew testDebugUnitTest
```
